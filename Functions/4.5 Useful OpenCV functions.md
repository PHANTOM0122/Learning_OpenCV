#### Mat::setTo() 함수
<pre>
<code>
Mat& MAt::SetTo(InputArray value, InputArray mask = noArray());
</code>
</pre>
* value : 행렬 원소에 설정할 값
* mask : 마스크 행렬, **마스크 행렬의 값이 0이 아닌 위치에서만 value값이 설정된다.**
* 반환값 : Mat 객체
